# Product Overview

## Python Function Testing Tool

An interactive tool for automatically discovering and testing Python functions in codebases. The tool provides dual interfaces (terminal and web) for comprehensive function testing with intelligent input generation.

### Core Features
- **Function Discovery**: Automatically scans Python files to find public functions and class methods
- **Intelligent Input Generation**: Uses LLM-powered analysis (OpenAI API) with rule-based fallbacks
- **Class Instance Support**: Interactive creation of class instances for function parameters
- **Multiple Testing Modes**: Manual input, auto-generation, and batch testing
- **Dual Interface**: Terminal-based and modern web interface options
- **Test Verification**: Manual verification of expected outputs with result tracking

### Secondary Components
- **Page Index System**: Document processing tool for extracting table of contents and hierarchical structures from documents
- **Sample Functions**: Comprehensive test suite including mathematical operations, string processing, class-based examples, and PyTorch neural network components

### Target Users
- Python developers testing function implementations
- QA engineers validating code behavior
- Educators demonstrating function testing concepts
- Researchers processing document structures
